# Frontend UI Assignment

## 📌 Overview
This project is a responsive web page built using vanilla HTML, CSS, and JavaScript based on the provided Figma design.

## 🚀 Features
- Fully responsive design (Desktop, Tablet, Mobile)
- Sticky header with smooth scroll behavior
- Interactive image carousel with hover zoom effect
- Clean and structured UI layout

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla)

## 📁 Project Structure
- index.html
- style.css
- script.js


## 🎯 Implementation Details
- Followed pixel-perfect design based on Figma
- Used semantic HTML for accessibility
- Optimized layout spacing and alignment
- Smooth transitions and hover effects applied

## 📌 How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
